QRFocus Privacy Policy

Last updated: July 23, 2025

Important: QRFocus prioritizes your privacy above all else. This app does not transmit personal information to external servers. All data is stored only on your device.

1. Introduction

This Privacy Policy explains what information QRFocus ("the App") collects, how it is used and protected. By using this app, you agree to this Privacy Policy.

2. Information We Collect

2.1 Information Directly Collected by the App

  • Scan History Data
    • Content of scanned QR codes
    • Scan date and time
    • QR code type (URL, text, Wi-Fi, etc.)
  • Generated QR Code Data
    • Content of generated QR codes
    • Generation date and time
  • App Settings
    • Vibration settings
    • History saving settings
    • Dark mode settings
    • Language settings

2.2 Information We Do NOT Collect

  • Personally identifiable information (name, email, phone number, etc.)
  • Location information
  • Contact information
  • Device-specific identifiers
  • Any other personal information

3. Required Permissions and Usage

3.1 Camera Permission

  • Purpose: QR code scanning functionality only
  • Access time: Only when the scan screen is open
  • Captured images are temporarily processed and not saved

3.2 Vibration Permission

  • Purpose: Haptic feedback when QR code scanning succeeds
  • Access time: Only during successful scans (can be disabled in settings)

3.3 Storage Permission (Android 10 and below)

  • Purpose: Export scan history to CSV files (user-initiated only)
  • Access time: Only when user explicitly chooses to export data
  • Data handling: Files are created in user-accessible storage as requested
  • Note: This permission is automatically granted on Android 11+ via scoped storage

4. Data Storage and Management

4.1 Data Storage Location

All data is stored locally on your Android device in the app's dedicated area (SharedPreferences). No data is transmitted to external servers.

4.2 Data Protection

  • Data is stored within Android's security sandbox
  • Other apps cannot access this data
  • Protected by your device's security settings (screen lock, etc.)

4.3 Data Deletion

  • Individual history items: Can be deleted individually from the history screen
  • All history: Can be deleted in bulk from the settings screen
  • All data is automatically deleted when the app is uninstalled

5. Third-Party Services

5.1 Libraries Used

This app uses the following open-source libraries, none of which collect user data:

  • Google ML Kit (barcode scanning)
  • CameraX (camera functionality)
  • ZXing (QR code generation)
  • Jetpack Compose (UI)

5.2 Advertising & Analytics

Current Status: This app currently does not display advertisements or use analytics tools.

Future Updates: We may introduce advertising and analytics features in future versions to improve app functionality and user experience. If such features are added:

  • We will update this Privacy Policy before implementation
  • Users will be notified within the app about these changes
  • Explicit consent will be obtained before data collection begins
  • Users will have the option to opt out of these features
  • Any third-party advertising or analytics partners will be listed in the updated policy

6. Data Sharing

This app does not share collected data with third parties, except in the following cases:

  • When you explicitly use sharing features (sharing QR code images, etc.)
  • When legally required (though we don't collect personal information anyway)

7. App Usage Requirements

This app is designed for general users. We recommend appropriate adult supervision for younger users who may need assistance understanding QR code content and safety.

8. Security

This app implements the following measures to ensure data security:

  • All data stored locally on device
  • No external communication by design
  • Compliance with Android security best practices
  • Requesting only minimal necessary permissions

9. Privacy Policy Changes

This Privacy Policy may be updated as needed. Important changes will be notified within the app. The last update date is listed at the top of this policy.

10. Your Rights (GDPR Article 12-23)

You have the following rights:

  • Right of Access (Article 15): You can view all stored data in the history screen
  • Right to Rectification (Article 16): You can modify scan history entries
  • Right to Erasure (Article 17): You can delete individual or all data at any time
  • Right to Data Portability (Article 20): You can export history data in CSV format
  • Right to Object (Article 21): You can change data collection settings at any time
  • Right to Withdraw Consent: You can disable data collection features in settings

10.1 Data Processing Legal Basis

Data processing is based on:

  • Legitimate Interest (Article 6(1)(f)): Core app functionality (QR scanning/generation)
  • Consent (Article 6(1)(a)): Optional features (history saving, vibration)
  • Contract Performance (Article 6(1)(b)): Essential app services
