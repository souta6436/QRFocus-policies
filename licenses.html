QRFocus ライセンス情報

QRFocusは、多くの優秀なオープンソースライブラリの恩恵を受けて開発されています。 これらのライブラリの開発者および貢献者の皆様に深く感謝いたします。

Android Jetpack Libraries

androidx.core:core-ktx, androidx.appcompat:appcompat, androidx.activity:activity-compose, androidx.navigation:navigation-compose

ライセンス: Apache License 2.0

著作権者: The Android Open Source Project

説明: Androidアプリケーション開発の基盤ライブラリ

Jetpack Compose

androidx.compose.ui:ui, androidx.compose.material3:material3, androidx.compose.material:material-icons-extended

ライセンス: Apache License 2.0

著作権者: The Android Open Source Project

説明: モダンなAndroid UIツールキット

CameraX

androidx.camera:camera-core, androidx.camera:camera-camera2, androidx.camera:camera-lifecycle, androidx.camera:camera-view

ライセンス: Apache License 2.0

著作権者: The Android Open Source Project

説明: カメラ機能を簡単に実装するためのライブラリ

Google ML Kit

com.google.mlkit:barcode-scanning

ライセンス: Google ML Kit Terms of Service

著作権者: Google LLC

説明: バーコード・QRコードのスキャン機能を提供

ZXing

com.google.zxing:core, com.journeyapps:zxing-android-embedded

ライセンス: Apache License 2.0

著作権者: ZXing Project

説明: QRコード生成ライブラリ

Accompanist

com.google.accompanist:accompanist-permissions, com.google.accompanist:accompanist-webview

ライセンス: Apache License 2.0

著作権者: Google LLC

説明: Jetpack Composeの追加機能を提供

Gson

com.google.code.gson:gson

ライセンス: Apache License 2.0

著作権者: Google Inc.

特別な謝辞

QRFocusの開発にあたり、以下のコミュニティとプロジェクトに特に感謝いたします：

  • Android Open Source Project - 素晴らしいプラットフォームとライブラリの提供
  • Google Developers - ML KitやJetpack Composeなどの革新的なツール
  • ZXing Project - QRコード処理の業界標準ライブラリ
  • オープンソースコミュニティ - 技術の発展と知識の共有

オープンソースソフトウェアなくして、QRFocusは存在しませんでした。
すべての貢献者の皆様に心より感謝申し上げます。
